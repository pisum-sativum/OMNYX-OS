# Code of Conduct

## Our Commitment

OMNYX is an open project. Everyone who contributes, opens issues, participates in discussions, or uses this codebase is part of this community.

We are committed to keeping this space respectful, inclusive, and focused on building something excellent.

---

## Standards

Behavior that makes this community work:

- Communicating clearly and respectfully
- Giving and receiving technical feedback professionally
- Focusing on the work, not the person
- Welcoming contributors at all experience levels
- Being honest about tradeoffs and limitations

Behavior that is not acceptable:

- Harassment, insults, or personal attacks
- Discriminatory language or behavior of any kind
- Sharing others' private information without consent
- Deliberate disruption of discussions or contributions
- Any conduct that would be considered inappropriate in a professional environment

---

## Scope

This code of conduct applies to all project spaces: GitHub issues, pull requests, discussions, and any other channel where this project is represented.

---

## Enforcement

If you observe unacceptable behavior, report it by opening a [GitHub Security Advisory](https://github.com/nagadevikona20max/omnyx/security/advisories/new) (for private reporting) or flagging the specific comment/issue to the maintainer.

Reports will be reviewed promptly. Violations will result in warnings, temporary restrictions, or permanent removal depending on severity.
